## Generative AI Practice for ITSQB

### Overview
This repository is a Generative AI practice activity for ITSQB, focused on AI-assisted rephrasing and improvement of software testing documentation.

### Purpose
- Apply Generative AI in a testing context  
- Improve clarity and quality of written testing artifacts  
- Support ITSQB learning and assessment requirements

### Scope
This repository contains documentation examples only. No production code is included.
